# Full Stack Cafe ☕
Full Stack Cafe is a web application built using Node.js, Express.js, MongoDB, and EJS.  
It allows users to sign up, log in, and browse cafe products.  
The project uses JWT authentication to secure user accounts.

## ✨ Features

- User Signup and Login
- JWT Authentication
- Secure Password Hashing using bcrypt
- Product Page
- Add to Cart functionality
- Protected Routes
- Responsive UI using EJS

- ## 🛠 Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- JSON Web Token (JWT)
- bcrypt.js
